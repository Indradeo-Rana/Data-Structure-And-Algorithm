An Applet is a small Java program that runs inside a web browser or an applet viewer. 
It is used for creating interactive web-based applications like animations, games, or graphics.

Applet runs on the client side.

It does not have a main() method.

It extends the java.applet.Applet or javax.swing.JApplet class.

Uses HTML to embed into a webpage.

It has special lifecycle methods: init(), start(), stop(), destroy().
